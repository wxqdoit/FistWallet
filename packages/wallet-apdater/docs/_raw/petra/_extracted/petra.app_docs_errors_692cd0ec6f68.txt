[H1] Errors
[P] When making requests to Petra API, you may receive an error. The following is a partial list of the possible errors and their corresponding codes:
[H2] Code 4000
[LI] Code: 4000
[LI] Name: No Accounts
[LI] Message: No accounts found.
[H2] Code 4001
[LI] Code: 4001
[LI] Name: User rejection
[LI] Message: The user rejected the request.
[H2] Code 4100
[LI] Code: 4100
[LI] Name: Unauthorized
[LI] Message: The requested method and/or account has not been authorized by the user.

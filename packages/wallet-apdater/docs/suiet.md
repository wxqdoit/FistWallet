# Suiet

## Supported chains
- Sui

## Recommended integration
- Use Suiet Wallet Kit (auto-detects wallets implementing Sui Wallet Standard)
- Wallet Standard connect: wallet.features['standard:connect'].connect()

## Notes
- Wallet Standard defines standard:connect and standard:events

# TronLink

## Supported chains
- TRON

## Injected provider
- window.tronLink
- tronLink.tronWeb is available after connection

## Connect
- tronLink.request({ method: 'tron_requestAccounts' })

## Notes
- Use tronWeb for signing and transactions after connection
